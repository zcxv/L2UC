# Glossary

## TO BE
 * Avatar - the game object representing a player
 * Player - the game object representing a avatar with player _direct control_ from player controller
 * NPC - the game object representing a non playable character
 * Item - the game object representing a dropped item
 * InventoryItem - the game object representing item in inventory

## Deprecated
 * Pawn → Avatar
 * User → Avatar
 * Player → Player
 * Monster → NPC
