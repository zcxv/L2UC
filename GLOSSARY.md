# Glossary

## TO BE
 * Avatar - the game object representing a player
 * Player - the game object representing an avatar under _direct control_ via the player controller
 * NPC - the game object representing a non-playable character
 * Item - the game object representing a dropped item
 * InventoryItem - the game object representing an item in the inventory

## Deprecated
 * Pawn → Avatar
 * User → Avatar
 * Player → Player or Avatar
 * Monster → NPC
